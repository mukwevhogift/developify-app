# **App Name**: Developify

## Core Features:

- User Authentication: Secure user authentication using Firebase Authentication with email/password and OTP verification.
- Wallet Management: Manage user wallet balance, add funds (mock integration), and withdraw funds using Firestore.
- Property Listings: Browse property listings fetched from a Firestore collection, displaying key details and media.
- Investment Flow: Invest in properties by deducting funds from the user wallet, updating funding progress in Firestore, and creating an investment record.
- Portfolio: Display user's active, completed, and underfunded investments.
- Real-time Notifications: Provide real-time updates on investment status and wallet transactions via Firestore listeners.
- Personalized Investment Tool: An AI-powered tool to suggest properties best suited to a user's individual goals.

## Style Guidelines:

- Primary color: Neon Blue (#63B5FF) for a modern fintech look.
- Background color: Graphite (#33363F), providing a dark, luxurious feel with high contrast.
- Accent color: Silver (#D3D4D8) for surfaces, emphasizing a minimalist design.
- Headline font: 'Space Grotesk', sans-serif for headlines, offering a techy feel.
- Body font: 'Inter', sans-serif for body text, providing a neutral, objective look.
- Minimalist icons with a neon blue outline to match the primary color scheme.
- Use of an 8pt grid system for consistent spacing and layout.
- Rounded cards with soft shadows to create a modern and visually appealing interface.
- Animated progress bars for loading states and financial transactions, enhancing user experience.
